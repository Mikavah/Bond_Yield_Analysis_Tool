# 固收收益率分析工具 V1（基础版）
## 功能说明
- 读取Excel中的国债/国开债收益率数据；
- 绘制收益率曲线（解决中文乱码问题）；
- 计算10年期国开-国债利差，并根据阈值给出交易提示。

## 使用方法
1. 安装依赖：pip install pandas matplotlib
2. 准备bond_data.xlsx数据文件；
3. 运行main.py，生成收益率曲线图片。

## 核心逻辑
- 解决matplotlib中文乱码：设置SimHei字体，且放在seaborn样式加载后；
- 利差计算：国开10Y收益率 - 国债10Y收益率；
- 阈值判断：利差>0.10→偏贵，>0.15→偏便宜。(仅简单示例，实际情况可以根据具体业务和相关策略进行修改和补充）
