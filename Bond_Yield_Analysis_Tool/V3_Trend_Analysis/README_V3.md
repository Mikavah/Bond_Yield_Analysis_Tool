# 固收收益率分析工具 V3（趋势分析+量化决策版）
## 功能升级（对比V2）
- 新增历史数据存储：自动保存每日数据到Excel，去重追加；
- 新增7天利差趋势分析：绘制趋势图，计算7天分位数；
- 量化交易决策：基于分位数给出“买入/持有/卖出”信号；
- 生成Excel版多工作表日报（贴合机构投研场景）。

## 使用方法
1. 安装依赖：pip install pandas matplotlib tushare openpyxl
2. 替换代码中的tushare token；
3. 运行main.py，生成趋势图、Excel日报、历史数据文件。

## 核心策略
- 利差分位数：(最新利差-7天最小值)/(7天最大值-7天最小值)；
- 决策逻辑：分位数>0.8→卖出，<0.2→买入，中间→持有。（仅为简化版本设计，后续需根据具体业务需求和策略进行修改补充）
