# 固收收益率分析工具 V2（自动日报版）
## 功能升级（对比V1）
- 自动拉取tushare实时国债/国开债数据（失败则用模拟数据兜底）；
- 生成Markdown格式交易日报；
- 自动保存收益率曲线图片到本地。

## 使用方法
1. 安装依赖：pip install pandas matplotlib tushare
2. 替换代码中的tushare token为自己的账号；
3. 运行main.py，自动生成日报图片和Markdown文件。

## 核心亮点
- 模拟数据兜底：避免因token/非交易日导致代码报错，后期可根据需求优化补充代码；
- 日报内容贴合固收交易员日常复盘。
