# 固收收益率分析与量化决策工具
## 项目背景
针对固收交易员日常工作中“手动整理收益率数据、分析利差趋势、生成交易日报”的痛点，开发的自动化分析工具，覆盖从数据获取到量化决策的全流程。

## 版本迭代（体现成长）
| 版本       | 核心功能                          | 业务价值                                  |
|------------|-----------------------------------|-------------------------------------------|
| V1 基础版  | 静态数据读取+收益率曲线+基础利差计算 | 解决手动画图、中文乱码问题                |
| V2 自动版  | 实时数据拉取+Markdown日报+兜底逻辑  | 自动生成日报，节省手动整理数据时间        |
| V3 进阶版  | 历史趋势+量化决策+Excel版日报      | 基于分位数给出交易信号，适配机构投研场景  |

## 技术栈
- 数据获取：tushare（中债登收益率数据）；
- 数据处理：pandas（清洗、计算、存储）；
- 可视化：matplotlib（收益率曲线+趋势图）；
- 报告生成：Markdown/Excel（贴合职场汇报）。

## 核心业务逻辑
1. 利差分析：10年期国开债-国债利差（固收交易核心指标）；
2. 量化决策：基于7天利差分位数判断交易信号；
3. 容错设计：实时数据拉取失败时自动切换模拟数据。

## 适用场景
- 固收交易员日常复盘、生成交易日报；
- 固收量化岗基础策略回测、信号生成；
- 投研团队收益率趋势分析、汇报。
